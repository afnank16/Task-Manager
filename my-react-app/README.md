# Task Manager Dashboard

A simple React + Tailwind CSS task management dashboard that allows users to create and organize tasks by status.

## Features

- Create new tasks
- Categorize tasks into:
  - My Tasks
  - Saved
  - Finished
  - Deleted
- Sidebar navigation
- Responsive dashboard layout
- Modal-based task creation form
- Task filtering by status
- Clean UI built with Tailwind CSS

## Technologies Used

- React
- JavaScript (ES6+)
- Tailwind CSS
